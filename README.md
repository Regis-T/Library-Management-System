# 📚 Library Management System

A simple command-line **Library Management System** built with **Python**. The application allows users to manage a collection of books by adding, borrowing, returning, searching, and listing books while tracking their availability.

---

## 📖 Project Overview

This project was developed to demonstrate the fundamentals of **Object-Oriented Programming (OOP)** in Python. It uses classes, objects, methods, loops, and conditional statements to simulate a basic library management system.

The application runs entirely in the terminal and does not require any external libraries or databases.

---

## ✨ Features

- 📖 Add new books
- 📚 Borrow available books
- 🔄 Return borrowed books
- 🔍 Search books by title
- 📋 Display all books
- 📊 Count available books
- 💻 Interactive command-line menu

---

## 🛠 Technologies

- Python 3
- Object-Oriented Programming (OOP)

---

# 📚 OOP Concepts Implemented

## Classes

- `Book`
- `Library`

## Objects

Book objects are created and stored inside the Library class.

## Encapsulation

Each book stores:

- Title
- Author
- Availability Status

## Methods

- `add_book()`
- `borrow_book()`
- `return_book()`
- `list_books()`
- `search_book()`
- `count_available_books()`

# ⚙️ Functionality

The program allows users to:

- Add books to the library
- Borrow available books
